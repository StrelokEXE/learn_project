# Лабораторная работа 2

## О чём проект
Разделили приложение на две части — frontend и backend, которые общаются через API.

## Что тут есть
- **backend** — отдаёт данные через API, есть автодокументация
- **frontend** — забирает данные с backend и показывает их в браузере

## Как запустить
1. Запусти backend
2. Запусти frontend
3. Открой frontend в браузере
